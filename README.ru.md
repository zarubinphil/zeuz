# zeuz — фабрика агентных workflow-систем

**Зевс** — фабрика: из идеи рождает полноценную агентную workflow-систему класса Фемида/Мнемозина. Проводит спеку через шесть мастеров-изобретателей (русские учёные, роль = характер) и закладывает конституцию в каждую рождаемую систему.

## Поток

```
«нужен новый workflow/система»
   → Стадия 0: Лобачевский + /grill-me  (спека: цель · вход+объём · инвариант полноты · ограничения)
   → Workflow({ name: 'zeuz-pipeline', args: спека })
       0 Observe (abtop, CTX-gate) → 1 Марков (кастинг душ) → 2 Шухов (архитектура + ворота)
       → 3 Котельников (токеносбережение + модель по функции) → 4 DAG artifact
       → 5 Лебедев (пишет файлы) → 6 Зворыкин (испытание → вердикт)
   → Зевс: Graphify (если знание) · зеркало Codex+VPS · сдача
```

## Раскладка (схема Афины)

| Путь | Что |
|---|---|
| `CLAUDE.md` | тонкий роутер: карта, автозапуск, конвенции |
| `rules/best-practices.md` | **конституция** — закладывается в каждую рождаемую систему |
| `specs/00-roadmap.md` | поток фабрики + команда (читать первым) |
| `agents/*.md` | 6 душ-мастеров + `zeuz.md` (надзиратель) |
| `workflows/zeuz-pipeline.js` | активный workflow с воротами в коде |
| `docs/decisions/` | ADR |
| `smoke/smoke.sh` | гейт: `node --check` + grep инвариантов |
| `runs/` | runtime (observability, DAG; gitignored) |
